# 🧠 AI Health Chatbot (Web)

An **AI-powered health chatbot** built with **Streamlit**.  
Enter your basic info and symptoms, and the model will suggest a likely health condition and precautions.  

⚠️ **Disclaimer**: This project is for **informational and educational purposes only**. It is **not a substitute** for professional medical advice, diagnosis, or treatment.  

---

## 🚀 Live Demo
👉 [Try the App](https://ai-health-chatbot-bo43szgap2qmekrzx37k8h.streamlit.app)  

---

## ✨ Features
- Enter **Name, Age, Gender, Symptoms**
- Get possible **disease predictions**
- Receive **precautionary advice**
- Interactive **Streamlit web UI**
- Uses **pre-trained dataset** (4920 rows × 132 columns, 131 features)
